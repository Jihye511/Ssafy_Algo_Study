## 📌Rule
[ :school:  오프라인]
- 매주 목요일 오후 7:30 - 9:30
- 매주 월요일마다 희망자 신청
- 유성온천 / 충남대 스터디룸 대여 (참여자들끼리 엔빵)
- 2인 1조로 한 문제 가져오기 
- 단, 참여 안 했을 경우 해당 문제는 자율
